## 4교시
- zip
  - 행렬을 세로로 묶음
  - 재사용이 안됨, 한번 쓰면 사라짐
  ```python
  a= [1,2]
  b= [3,4]
  x= zip(a,b) #[(1,3),(2,4)]
  arr=[[1,2,3],[4,5,6],[7,8,9]]
  x= zip([x for x in arr])
  print(x) #<zip object at ~~~>
  print(*x) 
  ```
